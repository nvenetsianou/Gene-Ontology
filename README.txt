### This is a web application within the context of the course "Web & Bioinformatics Application Architecture", UOA, Department of Biology, MSC "Bioinformatics - Computational Biology". 
First create the database according to the instructions inside the "MySQL" folder. 
Then, run code inside goDB folder. 
	1. Spring Boot Framework is used to create this Web App. 
	2. Spring Security is used for roles' creation and accessibility configuration.  
	3. Thymeleaf offers templates for HTML, CSS and Javascript.
